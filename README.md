Scanner digunakan untuk membaca input dari pengguna (dari keyboard).
LatihansoalUKL3 adalah nama kelas utama dari program.
main() adalah method utama yang dijalankan pertama kali saat program dijalankan.
Program akan menampilkan pesan “Masukkan Bilangan :”.
Pengguna mengetik angka, lalu nilainya disimpan ke variabel bilangan1.
bilangan1 % 2 → operasi modulus, yaitu mencari sisa hasil bagi 2.
Jika hasilnya 0, maka bilangan genap.
Jika hasilnya tidak 0, maka bilangan ganjil.
Jika bilangan1 % 2 == 0 → tampil pesan "Bilangan Genap"
Jika tidak → tampil pesan "Bilangan Ganjil"
Scanner dapat ditutup dengan masukan.close();
